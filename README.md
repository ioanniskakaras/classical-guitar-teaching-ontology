# Classical Guitar Teaching Ontology

This repository contains the OWL ontology for classical guitar instruction.

## Contents
- `ontology/ClassicalGuitarOntology.owl` — main ontology file

## Description
The ontology models educational knowledge related to classical guitar teaching, including learning objectives, instructional practices, technical skills, exercises, and assessment-related concepts.

## Purpose
This repository supports the documentation, sharing, and academic reuse of the ontology developed for research on ontology-based music education and low-code educational applications.
